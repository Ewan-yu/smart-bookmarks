# Smart Bookmarks 项目结构

## 目录结构

```
smart-bookmarks/
├── manifest.json           # 扩展配置（Manifest V3）
├── package.json            # 项目配置
├── tailwind.config.js      # Tailwind CSS 配置
├── docs/                   # 项目文档
├── icons/                  # 图标资源（16/48/128 尺寸）
├── scripts/                # 构建和打包脚本
└── src/
    ├── background/         # 后台脚本（Service Worker）
    │   └── background.js   # 消息处理、AI分析、链接检测
    ├── content/            # 内容脚本
    │   └── collector.js    # 网页信息采集
    ├── popup/              # 主界面（全屏标签页）
    │   ├── popup.html      # 主界面 HTML
    │   ├── popup.js        # 应用入口（3722 行）
    │   ├── modules/        # 功能模块（20个）
    │   ├── utils/          # 工具函数
    │   ├── ui/             # UI 组件和渲染器
    │   └── tailwind.*      # 样式文件
    ├── options/            # 设置页面
    │   ├── options.html    # 设置页 HTML
    │   └── options.js      # 设置页逻辑
    ├── db/                 # 数据库操作
    │   ├── indexeddb.js    # IndexedDB 封装（v2）
    │   └── migration.js    # 数据库迁移
    ├── api/                # API 调用
    │   └── openai.js       # OpenAI 兼容接口
    └── search/             # 搜索模块
        ├── local-search.js # 本地搜索
        └── ai-search.js    # AI 语义搜索
```

## 通信机制

模块间通过 `eventBus` 进行通信：

```javascript
// 发布事件
eventBus.emit(Events.BOOKMARK_ADDED, data);

// 订阅事件
eventBus.on(Events.BOOKMARK_ADDED, (data) => {
  // 处理事件
});
```

### 主要事件分类

- **书签事件**: BOOKMARKS_LOADED, BOOKMARK_ADDED, BOOKMARK_UPDATED, BOOKMARK_DELETED
- **导航事件**: NAVIGATION_CHANGED, FOLDER_SELECTED, BREADCRUMB_CHANGED
- **搜索事件**: SEARCH_PERFORMED, SEARCH_CLEARED, SEARCH_RESULTS_UPDATED
- **AI 分析事件**: ANALYSIS_STARTED, ANALYSIS_PROGRESS, ANALYSIS_COMPLETED
- **UI 事件**: CONTEXT_MENU_ACTION, KEYBOARD_ACTION, DIALOG_OPENED, DIALOG_CLOSED

## 技术栈

- **Manifest V3** - Chrome/Edge 最新扩展标准
- **ES Module** - 原生模块化，无打包器
- **IndexedDB** - 本地数据持久化
- **Tailwind CSS** - 样式框架
- **OpenAI API** - AI 分类和语义搜索
