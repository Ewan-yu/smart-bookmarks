# 变更日志

本文档记录 Smart Bookmarks 浏览器扩展的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [2.0.1] - 2026-03-16

### 修复
- 修复右键菜单不显示的问题
  - 移除旧的 ContextMenuRenderer，统一使用 contextMenuManager 模块
  - 修复 init() 函数的 DOM 加载时序问题
  - 修复 show() 方法中 _calculatePosition() 导致 display 被重置的问题
- 移除 popup.js 中冗余的全局 contextmenu 事件监听器

### 重构
- popup.js 模块化改进
  - 移除 getContextMenuItems() 函数（由模块内部管理）
  - 简化右键菜单相关代码

## [2.0.0] - 待定

### 新增功能

#### 架构重构
- 完整的模块化架构重构（Phase 4）
- UI 组件库提取到独立模块
- 核心业务逻辑模块化
- 统一的状态管理系统
- 模块间通信优化

#### AI 分类优化
- AI 分类提示词精简优化（从 550 行降至 200 行）
- 分类建议质量提升，优先创建具体技术分类
- 标签建议同时返回
- 放错目录检测和移动建议
- 保存后自动调用浏览器 API 移动书签
- 分类建议预览功能

#### 文件夹管理
- 新建文件夹功能
- 文件夹重命名功能
- 文件夹删除功能（带确认）
- 文件夹合并功能
- AI 智能合并建议
- 拖拽排序文件夹

#### 数据库升级
- IndexedDB v2 架构
- 数据库迁移框架
- 完善的错误处理和恢复机制
- 数据库版本管理

#### 链接检测增强
- HEAD 请求优先，失败降级到 GET
- uncertain 状态支持
- 断点续检功能
- 进度暂存和恢复
- 详细的失效原因显示

#### 搜索功能
- 本地全文搜索
- AI 语义搜索
- 混合搜索模式
- 高级搜索语法（AND/OR/NOT）
- 搜索结果高亮

#### 数据导入导出
- JSON 格式导入导出
- HTML 格式导出
- CSV 格式导出
- Markdown 格式导出
- 数据完整性验证

#### 用户体验
- 快捷键支持（Ctrl+Shift+B）
- 进度条实时显示
- 任务面板（可展开/收起）
- Toast 通知提示
- 确认对话框
- 加载状态指示器
- 响应式设计

### 改进优化

#### 性能优化
- 分批处理大量书签
- 虚拟滚动优化（如实现）
- 请求频率控制
- 内存占用优化
- 加载速度提升

#### 错误处理
- 网络错误自动重试
- 优化的错误提示
- 错误日志记录
- 优雅降级策略
- 用户友好的错误消息

#### 数据同步
- 浏览器收藏夹双向同步
- Content Script 页面信息采集
- Favicon 自动获取
- 元数据自动提取
- 冲突检测和解决

### Bug 修复

- 修复 AI 分类建议不理想的问题（盲目使用"已导入"等宽泛分类）
- 修复保存后书签未实际移动到对应目录的问题
- 修复 `handleGetBookmarks()` 返回 `tags: []` 硬编码空数组的问题
- 修复 `PAGE_INFO_COLLECTED` 消息未处理的问题
- 修复分析因非取消原因中断后进度丢失的问题
- 修复文件夹操作的各种 UI 问题

### 已知问题

- `checkSingleLink()` 单条链接检测功能待实现（P2）
- 大量书签（5000+）性能待验证
- 深色模式待实现
- 定期自动检测功能未暴露 UI 入口

### Breaking Changes

- 数据库架构从 v1 升级到 v2，首次使用会自动迁移
- AI 分类提示词格式变更，可能影响分类结果
- 部分模块接口变更，如有自定义集成需更新代码

## 版本说明

### 版本格式

- **主版本号 (Major)**: 不兼容的 API 变更
- **次版本号 (Minor)**: 向下兼容的功能新增
- **修订号 (Patch)**: 向下兼容的 Bug 修复

### 变更类型

- **新增功能**: 新增的功能
- **改进优化**: 现有功能的改进
- **Bug 修复**: 问题修复
- **Breaking Changes**: 不兼容的变更
- **已知问题**: 已知但未修复的问题
- **安全更新**: 安全相关的修复
