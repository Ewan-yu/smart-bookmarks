```javascript
// 模块发送事件
eventBus.emit(Events.BOOKMARK_DELETED, { ids, count });

// popup.js 监听事件
eventBus.on(Events.BOOKMARK_DELETED, ({ ids, count }) => {
  renderBookmarks();
  updateFooterStats();
  Toast.success(`已删除 ${count} 个书签`);
});
```

## 🐛 修复的问题

### Bug #1: 书签加载失败
**问题**：bookmarkManager 发送 `GET_ALL_DATA`，background 只支持 `GET_BOOKMARKS`
**修复**：修改 bookmarkManager 使用正确的消息类型
**提交**：`649c94e`

### Bug #2: 更新书签 undefined 错误
**问题**：bookmark 可能为空对象，访问 `bookmark.title` 报错
**修复**：使用可选链操作符 `bookmark?.title`
**提交**：`020f0f0`

### Bug #3: 消息字段名不匹配
**问题**：popup.js 发送 `data` 字段，被错误解析为 `updates`
**修复**：同时支持 `data` 和 `updates` 两种格式
**提交**：`70b529b`
