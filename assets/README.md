# 商店素材目录

此目录用于存放发布到 Chrome Web Store 和 Edge Add-ons 所需的素材文件。

## 目录结构

```
assets/
  ├── icons/          # 扩展图标
  ├── marquee/        # 宣传图片
  └── screenshots/    # 功能截图
```

## 需要准备的素材

### 1. 图标 (icons/)

- [ ] `icon16.png` - 16x16 像素
- [ ] `icon48.png` - 48x48 像素
- [ ] `icon128.png` - 128x128 像素

**注意:** 当前项目根目录的 `icons/` 文件夹已有基本图标，可以考虑优化设计后放在这里。

### 2. 宣传图片 (marquee/)

- [ ] `marquee-1280x800.png` - 1280x800 像素（推荐）
- [ ] `marquee-640x400.png` - 640x400 像素（备选）

### 3. 功能截图 (screenshots/)

- [ ] `screenshot-1-main.png` - 主界面展示
- [ ] `screenshot-2-ai.png` - AI 分析功能
- [ ] `screenshot-3-check.png` - 链接检测功能
- [ ] `screenshot-4-search.png` - 搜索功能
- [ ] `screenshot-5-settings.png` - 设置页面

## 详细要求

请参阅 `docs/STORE_ASSETS.md` 了解详细的素材要求、尺寸规格和设计建议。

## 创建工具推荐

- **Figma** - https://www.figma.com/ - 在线设计工具
- **Canva** - https://www.canva.com/ - 简单易用的设计平台
- **Snipping Tool** - Windows 自带截图工具
- **TinyPNG** - https://tinypng.com/ - 图片压缩优化
