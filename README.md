# Smart Bookmarks - 智能收藏夹管理插件

> 一款智能的浏览器收藏夹管理插件，支持 AI 分类、失效链接检测、快速搜索

## 功能特性

### AI 智能分类
- 自动分析收藏内容，智能推荐分类
- 支持 OpenAI 兼容 API（可自定义服务商）
- 同时返回分类建议和标签建议
- 检测放错目录并给出移动建议

### 失效链接检测
- 一键扫描失效链接，轻松清理
- HEAD 请求优先，失败降级到 GET
- 支持断点续检和进度暂存
- 详细的失效原因显示（404/超时/DNS失败等）

### 智能搜索
- 本地全文搜索
- AI 语义搜索（需配置 API）
- 混合搜索模式
- 高级搜索语法（AND/OR/NOT）

### 数据管理
- 本地导入导出（JSON/HTML/CSV/Markdown）
- 多标签管理
- IndexedDB 本地存储，隐私安全

### 快捷操作
- `Ctrl+Shift+B` - 打开主界面
- `Ctrl+K` / `Ctrl+F` - 聚焦搜索框
- `F2` - 编辑选中项
- `Delete` - 删除选中项

## 安装方法

### Chrome/Edge

1. 下载或克隆本项目
2. 打开浏览器扩展管理页面 (`chrome://extensions/`)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目根目录

## 配置 AI 功能

1. 点击插件图标，打开主界面
2. 点击右上角"设置"按钮
3. 在 AI 配置区域填写：
   - API 地址（如 `https://api.openai.com/v1`）
   - API 密钥
   - 模型名称（如 `gpt-3.5-turbo`）
4. 保存配置

## 使用指南

### AI 分类
1. 点击"一键分析"按钮
2. 等待 AI 分析完成
3. 查看分类建议，确认后应用

### 失效检测
1. 点击"失效检测"按钮
2. 确认开始检测
3. 实时查看检测进度
4. 检测完成后，失效链接自动移至"待清理"分类
5. 确认后一键删除

### 搜索收藏
- 直接在搜索框输入关键词
- 配置 AI 后可使用语义搜索
